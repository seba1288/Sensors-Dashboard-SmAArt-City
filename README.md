# Sensors Dashboard - SmAArt City

A dashboard for visualizing and managing sensor data for a smart city environment.

## Getting Started

Follow these steps to clone the repository and run the project locally.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/seba1288/Sensors-Dashboard-SmAArt-City.git
   cd Sensors-Dashboard-SmAArt-City
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   *(If `npm run dev` doesn't work, you can also start it directly with `npx vite`)*

4. **Open in browser:**
   The application should now be running locally. Open your browser and navigate to the URL provided in your terminal (usually `http://localhost:5173` or `http://127.0.0.1:5174`).
